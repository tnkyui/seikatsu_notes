# SEIKATSU Notes

## サイト概要
### サイトテーマ
買った食品や調味料、消耗品などを登録し、使用度を更新してキッチン内の在庫を管理したり、
やることリストを作成して家事の進捗を管理したりする、リスト管理アプリです。

### テーマを選んだ理由
昨今、一人暮らしをしている方や、夫婦共働きをしている世帯が増えています。
働きながら家の仕事を完璧にこなすことは実際難しく、例えば食品を誤って多く買ってしまったり、結果ダメにしてしまったりすることもします。
お風呂掃除やトイレ掃除をうっかり忘れているだけで、部屋中に悪臭が漂っていたりします。
スケジュール帳やメモアプリに書くほどでもない、だけどいつも「うっかり」失敗してしまう生活のアレコレを、まとめられるアプリがあればと思いこのテーマに決めました。

### ターゲットユーザ
 - 一人暮らしをしている方
 - 共働きの世帯

### 主な利用シーン
 - 朝起きた時にやることリストをチェック
 - スーパーでのショッピング中に何を買うか決める時
 - 料理中に使い切った、あるいは残り僅かになった食材や調味料があった時
